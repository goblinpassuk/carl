🔐 How It Works

No registration required! The domain itself acts as the salt:

  1. Domain is hashed → challenge
  2. YubiKey PRF(challenge) → master key
  3. HKDF(master key, domain+salt) → password entropy
  4. Entropy → formatted password

✅ Why It's Truly Stateless

  • No registration ceremony
  • No credential storage
  • No browser state needed
  • Domain is the identifier
  • Same YubiKey anywhere = same passwords

🔑 The Math

challenge = SHA256("github.com:user:v1")
master_key = YubiKey.HMAC(challenge)
password = HKDF(master_key, salt, length)

Same inputs = Same outputs. Always.
