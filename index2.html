🔐 Truly Stateless Design

No registration. No credential storage. No setup.

The YubiKey has a factory-programmed master secret that is used for HMAC-Challenge-Response in slot 2 by default. We use WebAuthn with an empty allowCredentials list, which prompts the browser to find ANY discoverable credential on the YubiKey.

🔄 The Password Derivation

challenge = SHA256(domain + username)
response = YubiKey.PRF(challenge)
master_key = response
password = HKDF(master_key, salt, length)
